# RUN.txt - How to Run Dodge AI

## QUICK START (30 seconds)

1. Get API key from https://openrouter.ai (free signup)

2. Set environment variable:
   export OPENROUTER_API_KEY="your_key_here"

3. Run:
   python3 app.py

4. Open browser:
   http://localhost:5001

## THAT'S IT! 🚀

## DETAILED STEPS

### Step 1: Get OpenRouter API Key (2 min)
- Go to https://openrouter.ai
- Click "Sign Up"
- Go to Settings → API Keys
- Click "Create Key"
- Copy the key (starts with sk-or-v1-...)

### Step 2: Open Terminal

Navigate to project folder:
cd /Users/abhimanyurajjha/Downloads/files

### Step 3: Set API Key

Paste this command:
export OPENROUTER_API_KEY="paste_your_key_here"

Replace "paste_your_key_here" with your actual OpenRouter key

### Step 4: Start App

You should see:

### Step 5: Open Browser

Go to: http://localhost:5001

Done! Now ask questions about your data.

## TROUBLESHOOTING

### Error: "OPENROUTER_API_KEY not set"
→ Did you run the export command? Try again:
export OPENROUTER_API_KEY="your_key"

### Error: "Port 5001 already in use"
→ Kill existing process:
pkill -f "python3 app.py"
Then restart: python3 app.py

### Error: "No such table: business_partners"
→ Database not created. Run:
python3 ingest.py

### Slow responses (5+ seconds)
→ Normal! The free model takes 3-5 seconds
→ To speed up, upgrade OpenRouter to paid tier

### Graph not showing
→ Check browser console (F12)
→ Verify: curl http://localhost:5001/api/health
Should return: {"status": "ok"}
